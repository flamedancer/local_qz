改少数字体颜色, 可以用, <font color=red> 某某 </font>, 但font关键字己过时, 尽量用 style: <span style="color:red; font-size: 1.2em" > 某某 </span>

常用的颜色: red 红, green 绿, blue 蓝, orange 橙, purple 紫, yellow 黄, cyan 青/蓝绿, violet 紫红, teal 灰绿/鸭绿

<br> : 手动分行
<p> : 分段落
&nbsp; : 强制空格(英文/半角),
<img src="image_ui/notice.png" border=1 style="width: 50%" > : 插入图片

有数字排序的列表 (可以多层次嵌套)
<ol>
<li> ... </li>
</ol>

无数字的列表 (可以多层次嵌套)
<ul>
<li> ... </li>
</ul>

改字体大小时, 单位用 em (字母m的宽度), 比 px (屏幕上的点), 可能会让 ipad, iphone 显示得更一致: <span style="font-size: 1.3em"> 大号 </span>

尽量用CSS样式文件 ipad.css / iphone.css 中己定义的样式类的名字，可以省去再分别调整/设置 ipad, iphone上的字体大小.

DIV 对公告排版很重要, 正文中的 <div> 与 </div> 必须一一配对。
